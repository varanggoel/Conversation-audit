# Edoofa Conversation Audit Tool

This workspace contains a Streamlit app for auditing WhatsApp-style counselor conversations.

## Run locally

```bash
streamlit run app.py
```

## What it does

- Parses exported WhatsApp conversations from text files
- Applies an Edoofa-specific audit framework
- Produces findings with evidence, why-it-matters, and recommendations
- Supports uploading one or more conversations and reviewing cross-conversation patterns
